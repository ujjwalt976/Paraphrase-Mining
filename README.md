# 🔍 Paraphrase Mining using Sentence Transformers

A Natural Language Processing (NLP) project that identifies semantically similar sentence pairs using **Sentence Transformers**. This notebook demonstrates how transformer-based embeddings can detect paraphrases by understanding sentence meaning rather than relying on exact word matching.

---

## 📌 Project Overview

Paraphrase Mining is the process of finding sentences that convey the same meaning using different words. This project uses a pre-trained Sentence Transformer model to generate sentence embeddings and compute semantic similarity scores.

The notebook is designed for beginners and demonstrates the complete workflow, from loading data to identifying highly similar sentence pairs.
